# 🎮 Xbox App – Meu App de Apresentação

Aplicativo React Native com tema Xbox, desenvolvido com Expo e navegação Drawer.

## Telas

| Tela | Descrição |
|------|-----------|
| **Home** | Logo Xbox, mensagem de boas-vindas e botão Começar |
| **Sobre** | Nome do aluno, curso e descrição do app |
| **Serviços** | Game Pass, Gears of War, Forza, Cloud Gaming, Day One Games |
| **Galeria** | Emojis representando jogos Xbox com legendas |
| **Contato** | E-mail, telefone, cidade e botão Voltar para Home |

## Tecnologias

- React Native
- Expo
- React Navigation (Drawer)
- react-native-gesture-handler
- react-native-reanimated
